# Medical Insurance Cost Prediction

## Problem Statement
Predict annual medical insurance charges using demographic and lifestyle data.

## Dataset
Medical Cost Personal Dataset (Kaggle)

## Features
- Age
- Gender
- BMI
- Children
- Smoker
- Region

## Techniques Used
- Data Cleaning & EDA
- Feature Engineering
- Log Transformation
- One-Hot Encoding
- Regression Models
  - Linear Regression
  - Ridge & Lasso
  - Gradient Boosting
- Cross Validation

## Evaluation Metrics
- RMSE
- MAE
- R² Score

## Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Results
Gradient Boosting Regressor achieved the best performance with improved R² score.

## How to Run
1. Clone the repo  
2. Install requirements  
3. Run the notebook in Jupyter / Google Colab
